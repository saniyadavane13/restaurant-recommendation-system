🍽 Restaurant Recommendation System



A smart web-based restaurant recommendation system built using Python, Flask, Pandas, and Scikit-learn.

This project helps users discover restaurants based on  restaurant name, cuisine type, and preferences using machine learning techniques.





&#x20;🚀 Features



✅ Search by Restaurant Name

✅ Search by Cuisine Type

✅ Case-insensitive Smart Search

✅ Personalized Restaurant Recommendations

✅ Displays Rating and Cost for Two

✅ Clean Responsive UI

✅ Flask Web Application

✅ Real-world Restaurant Dataset Support







🛠 Tech Stack



\* Python

\* Flask

\* Pandas

\* Scikit-learn

\* HTML

\* CSS

\* Jinja2 Templates

⚙️ Installation \& Run



1️⃣ Clone Repository



bash

git clone https://github.com/saniyadavane13/restaurant-recommendation-system.git

cd restaurant-recommendation-system



2️⃣ Install Dependencies



bash

pip install flask pandas scikit-learn





3️⃣ Run Application



bash

python app.py





4️⃣ Open Browser

http://127.0.0.1:5000
